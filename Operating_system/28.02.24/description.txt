This is write on file. read, write and open.
1. Write a program to learn read() from keyboard file and write() into monitor file.
2. write a program to write into a normal(conventional) file using open(), read() and write()
3. write a program to create a named pipe using mkfifo()
4. write a program to write into a named pipe what is read from keyboard using open(), read(), write()
5. write a program to read from a named pipe and then display to monitor using open(), read(), write()
6. write a program to read from a named pipe and then display to monitor using open(), read(), write();

7. write a chat room using named pipe.
